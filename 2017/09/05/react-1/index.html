react简介

1. 专注视图层

react不是完整的MVC/MVVM框架，只专注于提供清晰简洁的view层解决方案。

2. virtual DOM

提升性能，减少对传统DOM的操作。
方便和其他平台集成，react可以映射成对应的原生控件。

3. 函数式编程

函数式编程是react的精髓。

JSX语法

1. JSX的由来

JSX将html语法直接加入到js代码中，再通过翻译器转换到纯js后由浏览器执行。
JSX并不是强制选项，只是让代码更加直观并易于维护。

2. JSX基本语法

定义标签时，只允许被一个标签包裹。
标签一定要闭合。
DOM元素首字母小写，组件元素首字母大写。
元素属性：class属性改为className for属性改为htmlFor
boolean属性：省略boolean属性值会导致JSX认为bool值设为true
自定义属性：使用data-前缀
js属性表达式：{}
html转义：dangerouslySetInnerHTML 避免react转义
